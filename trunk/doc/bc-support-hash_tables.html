package Tables is
The type Table represents an open hash table whose buckets may be formed by bounded, dynamic, or unbounded containers. Each table contains n buckets, wherein each bucket is a container of item/value pairs. To insert, remove, or locate a pair in the table, the operation first generates a hash value upon the item to select a specific bucket, and then the given operation is performed upon the selected container.
