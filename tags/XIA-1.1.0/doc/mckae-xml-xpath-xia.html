XIA (XPathInAda) is a native Ada implementation for querying XML documents using the XPath 1.0 query language.

Known limitations:

The core library function "id" is not yet implemented, as the XMLAda function Get_Element_By_ID is not yet implemented. Therfore invoking "id" will always result in an empty node-set.
